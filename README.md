# 📊 Attitude & Barrier Scores Analysis

## Overview

This project analyzes student attitudes toward research and perceived barriers using non-parametric statistical methods.

## Methods

* Likert scale recoding (1–3)
* Composite scores
* Mann–Whitney & Kruskal–Wallis tests

## Visualization

* Violin plots + boxplots + jitter
* Median highlighting
* Faceted demographics

## How to Run

```r
source("R/01_data_cleaning.R")
source("R/02_analysis.R")
source("R/03_visualization.R")
```

## Outputs

* High-resolution figures (SVG + TIFF)
* Publication-ready tables (DOCX)

## Notes

Optimized for non-normal data and academic publication.
